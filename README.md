# Hi, I'm Khan Munawar 👋

### 🛡️ Cybersecurity Analyst | VAPT Researcher | Network & Hardware Security

I am a passionate Cybersecurity and Vulnerability Assessment and Penetration Testing (VAPT) researcher. My work bridges the gap between software security, network infrastructure, and hardware prototyping. I focus on discovering vulnerabilities, automating security workflows with Python, and securing Industrial IoT (IIoT) environments.

---

### 💻 Technical Expertise

- **Security Testing:** VAPT, Network Auditing, Web Application Security, Hardware Exploitation
- **Tools & Platforms:** Kali Linux, Bettercap, Metasploit, Wireshark, WiFi Pineapple
- **Networking & Infrastructure:** Cisco, MikroTik, SonicWall, Firewalls, Routing & Switching
- **Languages & Automation:** Python (Security scripting, automation), Bash, C/C++ (Hardware prototyping)
- **Specialized Domains:** IIoT Security, Firmware Analysis, Vulnerability Research

---

### 🔬 Key Research & Projects

- **IIoT Gateway Vulnerability Research:** Conducted extensive security audits and penetration testing on industrial hardware (such as the MEERKAT Edge IIoT Gateway), uncovering high-severity flaws like Blind Command Injection and Credential Extraction.
- **Python Security Automation:** Developing lightweight, custom Python scripts for network scanning, credential auditing, and automated reconnaissance.
- **Network Architecture & Defense:** Designing and configuring secure network topologies utilizing enterprise-grade firewalls and routers to mitigate active threat vectors.
